Dashboard para Helpnei @ Fatec

2025 - Finalizado

No segundo semestre, desenvolvemos uma API completa voltada para a Helpnei e outras empresas parceiras. Minha contribuição foi na criação e estruturação do banco de dados, modelagem das tabelas com seus relacionamentos e no desenvolvimento dos endpoints responsáveis pelas funções de cadastro, consulta, atualização e exclusão de dados (CRUD). Também participei ativamente dos testes usando o Postman, garantindo que a API atendesse corretamente as necessidades dos clientes. Esse projeto me proporcionou muito aprendizado no back-end, reforçando meus conhecimentos em Node.js, Express e MySQL.
